# Phase 03.1 Plan 03: Shell — Brand Wordmark + Token Migration Summary

Re-skinned the global header, footer, and TRUST-06 disclaimer so every public route now opens on a cream canvas stamped with the VC·Finder brand wordmark (lime square + Geist-bold appName) and closes on a token-coherent footer — zero logic changes, i18n fully preserved, no more `text-neutral-*` leftovers anywhere in `src/components/site/`.

## What Shipped

**`src/components/site/header.tsx`** — brand wordmark + Geist display type:

- Lime 2×2px-rounded `<span>` (bg `var(--primary)`) marked `aria-hidden` precedes the translated `tCommon('appName')`. The dot is decorative; screen readers still announce the full brand name from the i18n string (FOUND-10 invariant).
- Wordmark link wrapped with `font-[family-name:var(--font-geist)]`, `text-lg`, `font-bold`, `tracking-[-0.3px]`, `text-[color:var(--foreground)]`.
- Shell container: `border-b border-[color:var(--border)] bg-[color:var(--background)]` — explicit cream + ink-10% hairline even when wrapped by another layout.
- Desktop nav: `text-[color:var(--muted-foreground)]` with `hover:text-[color:var(--foreground)] transition-colors` per link. Sets the shell hover convention Waves 4–6 can reuse.
- Logic preserved verbatim: async server component, `Promise.all([getLocale, getTranslations('header'), getTranslations('common')])`, navLinks array, UserMenu + MobileNav composition, locale-aware `/${locale}/sources|privacy|terms` hrefs.

**`src/components/site/footer.tsx`** — token migration + shell continuity:

- `<footer>` element: `border-t border-[color:var(--border)] bg-[color:var(--background)]` (was bare `mt-16 border-t`).
- Nav: `text-sm text-muted-foreground` (new muted inherit; matches header).
- Copyright line: `text-xs text-muted-foreground` (was `text-xs text-neutral-500`).
- Disclaimer placement (first element in flex column) preserved per D-04.7.
- All `getTranslations('footer')` calls + link hrefs unchanged.

**`src/components/site/disclaimer.tsx`** — one-line token swap:

- TRUST-06 paragraph: `text-xs text-muted-foreground leading-relaxed` (was `text-xs text-neutral-600 leading-relaxed`).
- `role="note"` + `t('disclaimerText')` preserved.
