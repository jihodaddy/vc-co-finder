# Phase 03.1 Plan 01: Foundation — Brand Tokens, Fonts, baseColor, DESIGN.md Summary

Replaced slate OKLCH tokens with the VC·Finder cream/ink/lime brand palette in `globals.css`, wired Pretendard Variable + Geist + Geist Mono via `next/font` and exposed them as Tailwind-consumable CSS variables, flipped `components.json baseColor` to `neutral`, and published `.planning/DESIGN.md` as the project-level source of truth for every downstream wave.

## What Shipped

**Brand tokens in `src/app/globals.css`** — All 17 semantic roles remapped to verbatim hex from `.design-import/ui.jsx TOKENS` (D-01.1..D-01.4):

- Surface: `--background #FBF9F4` (cream), `--foreground #14120E` (ink), `--card/--popover #FFFFFF`
- Signature: `--primary #CFEF3E` (lime), `--primary-foreground #14120E` (ink-on-lime), `--ring #CFEF3E` (lime halo)
- Muted: `--secondary/--muted/--accent #F4F0E6` (creamDeep), `--muted-foreground #78726A` (inkMute)
- Destructive: `--destructive #FFB5A0` (coral, per D-01.3)
- Hairlines: `--border/--input rgba(20, 18, 14, 0.10)` (ink @ 10% alpha)

Plus the utility namespace (D-01.5): `--radius-sm 4px`, `--radius-md 7px`, `--radius-lg 12px`, `--radius-pill 999px`; `--shadow-sm` subtle lift, `--shadow-md` card elevation; `--fw-regular/medium/semibold/bold` (400/500/600/700). Chart palette `--chart-1..5` (ink/lime/sky/coral/lilac) for Sparkline + Phase 6 time-series.

**Font pipeline in `src/app/fonts.ts`** — Three exports: `pretendard` (next/font/local, weight `45 920` explicit per RESEARCH Pitfall 2), `geistSans` (next/font/google), `geistMono` (next/font/google). All three `display: 'swap'` + fallback stack. Pretendard Variable woff2 (2.06MB) vendored at `public/fonts/PretendardVariable.woff2` — direct commit, not Git LFS (per RESEARCH Open Question #1).

**`@theme inline` font binding** — `--font-sans: var(--font-pretendard)` and `--font-mono: var(--font-geist-mono)` inside `@theme inline` (Tailwind v4 runtime-resolved variables; plain `@theme` would inline at build time and fall back to system fonts — RESEARCH Pitfall 1).

**`src/app/[locale]/layout.tsx`** — Imports `{ pretendard, geistSans, geistMono } from '@/app/fonts'`, applies all three `.variable` classes + `antialiased` on `<html>`.

**`components.json`** — `baseColor: "slate" → "neutral"` only. All other fields unchanged (aliases, style, rsc, tsx, cssVariables, iconLibrary). Wave 2's `npx shadcn add chart slider avatar` will now emit neutral-based starter templates.

**`.planning/DESIGN.md`** — 173-line project-level source of truth with all 5 required sections:

1. **Tokens** — full 17-row semantic table + chart palette + radius/shadow/weight scales
2. **Typography** — font stack + scale + 5 usage rules
3. **Primitives Inventory** — 16 shadcn primitives (13 existing + 3 Wave-2 additions) + 3 brand-domain metrics components
4. **Usage Rules** — semantic-tokens-only, lime-reserved, focus-halo standard, i18n, tabular-nums, destructive=coral, single-shadow
5. **v2 Deferred** — dark mode, full landing, Compare page, real data wiring, motion tokens, VRT, Storybook, custom icons
